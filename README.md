# Multiple Linear Regression - Assignment 2 

##  Overview
This repository contains two case studies based on Multiple Linear Regression.

The objective is to build prediction models, perform EDA, and evaluate model performance using real-world datasets.

---

##  Tasks Performed

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Building (Multiple Linear Regression)
- Model Comparison
- Model Evaluation (R², RMSE)
- Prediction Generation

---

##  Model Performance

Each case study includes:
- Multiple models with different feature sets
- Comparison based on R² score
- Selection of best model
